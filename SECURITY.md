# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in cmux, please report it by [opening an issue](https://github.com/ipdelete/cmux/issues/new?labels=security) with the **security** label.

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will review and respond as quickly as possible.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest release | ✅ |
| Older versions | ❌ |

Only the latest release receives security updates.
